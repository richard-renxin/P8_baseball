# 概要
数据集包含由惯用手分组的棒球运动员的中位数统计。每一组数据都包含惯用手、身高、体重、平均得分值、全垒打计数。
可视化图表分别解释了基于玩家惯用手分类的得分情况和身体量度。

# 设计
根据python中的分析结果得出两条结论,分别制作两张图表展示结论.                  
x轴作为离散变量,y轴采用双坐标轴反应两个连续变量.
采用柱状图或者散点图.



# 反馈和迭代

改进一:尝试制作分组柱状图,但是柱子本身是冗余信息,只保留柱子顶端即可.最后发现气泡图效果更好.
